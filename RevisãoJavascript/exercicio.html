Exercícios de Javascript

1. O que é um array em javascript?

a) Um conjunto, em lista, de dados
b) Uma função específica para ordenar dados
c) Um operador condicional
d) Um objeto de data

2. Qual a palavra-chave utilizada para definir uma função em javascript?

a) funtion
b) function
c) def function
d) func

3. Qual opção lista os comparadores na seguinte ordem: igual, diferente, maior que, menor que

a) ==, <>, <, >
b) =, !=, >, <
c) ===, !==, >, <
d) ===, , >, <

4. Qual o método utilizado para remover o último elemento de um array em Javascript?

a) push()
b) pop()
c) shift()
d) unshift()

5. Qual a palavra-chave utilizada para definir uma variável que não pode ser modificada em javascript?

a) const
b) let
c) var
d) static

6. Considere o seguinte código em Javascript:
let num = 49;
if (num > 5) {
num = num - 5
} else {
num = num + 9
}
console.log(num)


7. Considere o seguinte código em Javascript:
let x = 0
while (x < 5) {
x++
}
console.log(x)


8. Qual a palavra-chave utilizada para interromper uma execução de laço em repetição em Javascript?

a) break
b) continue
c) return
d) exit

9. O que é prompt em Javascript?

a) Uma função para criar um diálogo de confirmação
b) Uma função para criar um diálogo de alerta
c) Uma função para receber entrada do usuário
d) Uma função para exibir dados na tela

10. Considere o código abaixo:
let a = 100
let b = "ola"
console.log(a + b)

  • 4. Qual o método utilizado para remover o último elemento de um array em Javascript? Resposta correta: b) pop()
  • 5. Qual a palavra-chave utilizada para definir uma variável que não pode ser modificada em javascript? Resposta correta: a) const
  • 6. Qual é o resultado que será impresso no console? Resposta correta: 44
  • 7. Qual é o resultado que será impresso no console? Resposta correta: 5
